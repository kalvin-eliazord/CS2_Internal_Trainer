# CS2 Internal Trainer
An updated C++ dynamic-linked library aimbot for the CS2 game.

## Features
- Aimbot 
	- Smoothing
	- Target Locking
	- FOV
	- Head or Pelvis focus

- Pattern Scanning

## What is aimbot ?
This is a cheating functionality that will allow a player to aim at a target automatically.
